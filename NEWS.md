# outbreakdistr (development version)

* `emmdt()`: computes the exact final size distribution for a stochastic SIR epidemic using the embedded discrete-time Markov chain approach.
* `fsdistr()`: computes the exact final size distribution for a single-type
  stochastic SIR epidemic model.
* `fsdistr_mt()`: computes the exact joint final size distribution for a
  multi-type stochastic SIR epidemic model.
